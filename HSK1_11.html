Mode d'emploi

🔤 Vocabulaire

  • Audio : Cliquez sur le caractère chinois pour écouter la prononciation officielle.
  • Révélation : Le pinyin et le sens sont cachés pour vous aider à réviser. Cliquez sur les textes bleus pour les afficher.

💬 Dialogues

  • Lecture : "▶ Tout lire" joue le dialogue automatiquement.
  • Bulles (Tooltips) : Survolez les caractères chinois pour voir instantanément le Pinyin et la Traduction correspondante.

✍️ Écriture

  • Sélection : Cliquez sur un mot (ex: 什么, 时候, 几) pour voir comment le tracer.
  • Chercher : Utilisez la zone de recherche pour n'importe quel caractère, comme 去 ou 在.
  • Quiz : Écrivez le caractère de mémoire sans le modèle pour valider vos connaissances.
